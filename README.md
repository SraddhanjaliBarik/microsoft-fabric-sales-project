# microsoft-fabric-sales-project

## Project Overview

This project demonstrates an end-to-end Sales Analytics solution built using Microsoft Fabric. The solution follows the Medallion Architecture approach with separate Bronze, Silver, and Gold layers implemented using three different Lakehouses.

The project includes data ingestion, transformation, semantic modeling, dashboard creation, CI/CD implementation, and pipeline orchestration to deliver scalable and efficient business reporting solutions.

## Technologies Used

* Microsoft Fabric
* Power BI
* Lakehouse
* Dataflow Gen2
* Semantic Model
* ETL Pipelines
* CI/CD
* Star Schema
* DAX Measures

## Architecture

Implemented Medallion Architecture:

* Bronze Layer – Raw data ingestion
* Silver Layer – Cleaned and transformed data
* Gold Layer – Business-ready analytical data

## Features Implemented

* Created 3 different Lakehouses
* Built Dataflows for ETL processing
* Designed Star Schema data model
* Created Semantic Model for reporting
* Implemented ETL pipelines
* Configured CI/CD process
* Developed interactive Power BI dashboards
* Created DAX measures for business KPIs

## Key Metrics

* Total Sales
* Total Orders
* Sales per Customer
* Sales per Category

## Reports Created

### 1. Sales Overview Report

Provides high-level sales KPIs and overall business performance analysis.

### 2. Customer Insights Report

Analyzes customer behavior, purchasing trends, and customer-level sales performance.

### 3. Product Performance Report

Shows product-wise sales analysis, category performance, and top-performing products.

## Business Value

This project helps organizations:

* Monitor sales performance
* Analyze customer behavior
* Track product performance
* Improve decision-making using interactive analytics dashboards
